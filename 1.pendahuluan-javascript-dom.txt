DOM / document object model
DOM adalaj antarmuka pemrogaman untuk html yang mempersentasikan halaman web sehingga program dapat mengubah dan memanipulasi strukturnya "menurut mozila".

Document adalah document web kita
pada web browser terdapat document (isi dari websitenya)

Object sebuah nilai atau tipe data pada java script, object mempunyai properti, dan mempunyai method serta function.

Model repersentasi terhadap sesuatu

DOM adalah reprensentasi elemen html pada dokumen menjadi sebuah object.

didalam memori web, terdapat pohon hierarki dom (DOM Tree).

DOM
- antarmuka pemrograman bebasis objek yang mempersentasikan dokumen web
- DOM membuat seluruh komponen dari halaman web dapat diakses dan dimanipulasi
- komponen
    - elemen html
    - atribut
    - text
    - dll 
- dom dapat dimanipulasi (dibuat, diubah, dihapus) menggunakan javascript
